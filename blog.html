Pinjambuku

Laravel Bootstrap MySQL
Project Overview

Pinjambuku adalah aplikasi peminjaman buku yang memudahkan pengguna untuk mencari, meminjam, dan mengelola buku secara online. Aplikasi ini dirancang dengan fokus pada kemudahan penggunaan, keamanan, dan efisiensi proses peminjaman.

User Roles
Mahasiswa
  • Mencari dan melihat daftar buku
  • Meminjam buku
  • Melihat riwayat peminjaman
  • Mengelola profil
Admin
  • Mengelola data buku
  • Memverifikasi peminjaman
  • Mengelola kategori buku
  • Mengelola profil
Key Features
  • Sistem autentikasi yang aman
  • Pencarian buku yang mudah
  • Keranjang peminjaman
  • Manajemen profil pengguna
  • Katalog buku
  • Tracking status peminjaman
